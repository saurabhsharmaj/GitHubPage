Here are answers to your interview questions on Linux commands:

  1. How to login on remote Linux machine?

    • Use the ssh command:
      									
      
										
      
											bash
											
										
      
										
      
											ssh username@remote_host

  2. How to switch user?

    • Use the su command:
      									
      
										
      
											bash
											
										
      
										
      
											su - username
    • Or use sudo to run a command as another user:
      									
      
										
      
											bash
											
										
      
										
      
											sudo -u username command

  3. How to search if a process is running (like Tomcat)?

    • Use the ps command along with grep:
      									
      
										
      
											bash
											
										
      
										
      
											ps aux | grep tomcat

  4. How to check the PID of any process?

    • Use the pidof command:
      									
      
										
      
											bash
											
										
      
										
      
											pidof process_name
    • Or use ps combined with grep:
      									
      
										
      
											bash
											
										
      
										
      
											ps aux | grep process_name

  5. How to kill the Process?

    • Use the kill command with -9 option:
      									
      
										
      
											bash
											
										
      
										
      
											
												kill -9 PID

  6. How to change or check ownership of any file (chown)?

    • To check ownership:
      									
      
										
      
											bash
											
										
      
										
      
											
												ls -l filename
    • To change ownership:
      									
      
										
      
											bash
											
										
      
										
      
											sudo chown user:group filename

  7. How to set permissions on any file/folder (chmod)?

    • To change permissions:
      									
      
										
      
											bash
											
										
      
										
      
											
												chmod 755 filename

  8. How to check the logs?

    • Use commands like tail, cat, or less:
      									
      
										
      
											bash
											
										
      
										
      
											
												tail -f /var/log/syslog

  9. What is the use of the less command?

    • less is used to view the contents of a file one screen at a time:
      									
      
										
      
											bash
											
										
      
										
      
											less filename

  10. How to create a service in Linux and start/stop/restart it?

    • To create a service, you usually define a systemd unit file in /etc/systemd/system/.
    • To start a service:
      									
      
										
      
											bash
											
										
      
										
      
											sudo systemctl start service_name
    • To stop a service:
      									
      
										
      
											bash
											
										
      
										
      
											sudo systemctl stop service_name
    • To restart a service:
      									
      
										
      
											bash
											
										
      
										
      
											sudo systemctl restart service_name

  11. How to search a keyword in a file?

    • Use the grep command:
      									
      
										
      
											bash
											
										
      
										
      
											grep keyword filename

  12. How to check if a file is available in the system?

    • Use the find command:
      									
      
										
      
											bash
											
										
      
										
      
											find /path/to/search -name filename
    • Or use locate if the mlocate database is updated:
      									
      
										
      
											bash
											
										
      
										
      
											locate filename

  13. Networking connectivity commands:

    • nc (netcat):
      									
      
										
      
											bash
											
										
      
										
      
											nc -zv hostname port
    • ping:
      									
      
										
      
											bash
											
										
      
										
      
											ping hostname
    • telnet:
      									
      
										
      
											bash
											
										
      
										
      
											telnet hostname port
    • lsof:
      									
      
										
      
											bash
											
										
      
										
      
											lsof -i
    • nslookup:
      									
      
										
      
											bash
											
										
      
										
      
											nslookup hostname

  14. Firewall check command:

    • Use ufw for Uncomplicated Firewall:
      									
      
										
      
											bash
											
										
      
										
      
											sudo ufw status
    • Or use iptables:
      									
      
										
      
											bash
											
										
      
										
      
											sudo iptables -L

  15. Check the OS version:

    • Use the lsb_release command:
      									
      
										
      
											bash
											
										
      
										
      
											lsb_release -a
    • Or check the contents of /etc/os-release:
      									
      
										
      
											bash
											
										
      
										
      
											
												cat /etc/os-release

  16. OS troubleshooting commands:

    • top:
      									
      
										
      
											bash
											
										
      
										
      
											top
    • du:
      									
      
										
      
											bash
											
										
      
										
      
											
												du -sh /path/to/directory
    • df:
      									
      
										
      
											bash
											
										
      
										
      
											
												df -h
    • free:
      									
      
										
      
											bash
											
										
      
										
      
											free -h

  17. Crontab or job scheduling:

    • To edit the crontab:
      									
      
										
      
											bash
											
										
      
										
      
											crontab -e
    • To list current cron jobs:
      									
      
										
      
											bash
											
										
      
										
      
											crontab -l

  18. Network packet analysis and tracing:

    • tcpdump:
      									
      
										
      
											bash
											
										
      
										
      
											sudo tcpdump -i interface
    • traceroute:
      									
      
										
      
											bash
											
										
      
										
      
											traceroute hostname
